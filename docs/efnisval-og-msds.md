# Efnisval og MSDS

Hér verður lýst hvaða efni voru notuð í verkefninu og af hverju þau voru valin.

## Öryggi
Mikilvægt er að fylgja öryggisreglum við notkun Roland SRM-20 og við meðhöndlun efna.
