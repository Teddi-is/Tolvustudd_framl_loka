# Tölvustudd framleiðsla – Lokaverkefni

## Hópverkefni B-liður
Í þessu verkefni hönnum við, teiknum, undirbúum og fræsium mót í vax með Roland SRM-20. Verkefnið felur í sér hönnunarferli, verkefnastjórnun, CAM undirbúning, roughing og finishing toolpaths, öryggissjónarmið og skrásetningu á framkvæmd og afrakstri.
